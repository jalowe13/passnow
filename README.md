FastAPI, (ASGI to serve ) DynamoDB, and Python

```bash
# Setup
pip install fastapi
pip install uvicorn
docker pull amazon/dynamodb-local

# Running in GO
npm run startappgo
# Running in Python (Experimental)
npm run startapp
npm run start-uvicorn
```
